# 📊 Яндекс Вордстат — подписи точек + АППГ

Userscript для Яндекс Вордстат: подписывает все точки на графике истории запросов и добавляет колонку «Динамика к АППГ» в таблицу.

Работает только на вкладке **«Динамика»** — на других вкладках не активируется.

## ✨ Возможности

- **Подписи всех точек** на графике — число запросов (синий) и доля (красный)
- **Увеличенные точки** с белой обводкой для лучшей читаемости
- **Умное размещение подписей** — чередование вверх/вниз + итеративное отталкивание при наложении
- **Колонка АППГ** в таблице: динамика к аналогичному периоду прошлого года (▲/▼ %)
- **Сортировка по АППГ** — кликом по заголовку колонки (asc/desc)
- **Панель управления** — можно включить/выключить каждый элемент отдельно
- **Только вкладка «Динамика»** — на других вкладках скрипт не вмешивается


## 🔧 Установка

1. Установите расширение [Tampermonkey](https://www.tampermonkey.net/) для своего браузера
2. Нажмите на иконку Tampermonkey → **Создать новый скрипт**
3. Удалите весь дефолтный код и вставьте содержимое файла [`wordstat-labels.user.js`](/wordstat-labels.user.js)
4. Нажмите **Ctrl+S** 
5. Перейдите на [wordstat.yandex.ru](https://wordstat.yandex.ru), введите запрос, откройте вкладку **«Динамика»**

## 📋 Как пользоваться

- **График**: подписи появляются автоматически при открытии вкладки «Динамика»
- **Панель** (правый нижний угол): кнопки включения/выключения подписей, точек и колонки АППГ
- **Таблица**: колонка «Динамика к АППГ» добавляется справа, кликните по заголовку для сортировки
- При смене запроса / периода / устройства — всё пересчитывается автоматически
